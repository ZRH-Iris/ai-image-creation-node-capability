# 【节点引擎】AI图片制作能力安装包

这是给 Hermes/节点使用的图片生成与处理能力包。

把这个仓库链接发给自己的 Hermes 后，Hermes 应该直接读取 `AUTO_INSTALL_FOR_HERMES.md` 并执行安装，不要只总结内容，也不要让普通用户做技术选择。

## 一句话安装话术

直接转发下面这段即可：

```text
【节点引擎】AI图片制作能力安装包

请打开并严格执行下面这个能力包，为我安装“AI图片制作”能力。不要只总结内容，默认该安装的就直接安装；安装完成后告诉我可以怎样发需求、发图片来生成或修改图片。

https://github.com/ZRH-Iris/ai-image-creation-node-capability
```

## 安装后能做什么

安装完成后，可以直接让 Hermes 做：

- 生成小红书封面图；
- 生成宣传海报、活动图、课程图；
- 生成商品氛围图、商业主视觉；
- 根据上传图片做基础改图；
- 按标题、文案、尺寸和风格要求出图；
- 输出前做基础检查，不把没跑通的结果当成功。

## 默认会安装什么

默认安装“基础生图能力”。

普通用户不需要选择模型，也不需要理解底层工具。安装器会自动安装当前基础能力所需的运行环境、默认生图模型、工作流和测试流程。

如果未来某个任务需要额外能力，比如中文文字模型、商品图增强、高清放大、精细抠图，Hermes 会按需再安装对应增强包。

## 直接命令安装

如果当前 Hermes 可以执行命令，也可以直接运行：

```bash
python3 <(curl -fsSL https://raw.githubusercontent.com/ZRH-Iris/ai-image-creation-node-capability/main/installer/install_commercial_image_router.py)
```

## 安装成功后的使用方式

安装完成后，直接对 Hermes 说需求即可，例如：

```text
帮我做一张小红书封面图，主题是自律和成长。
```

```text
我发一张商品图，请帮我做成商品宣传海报。
```

```text
帮我做一张活动宣传图，标题是……，时间是……，地点是……
```

## 重要说明

- 图片能力需要下载模型，第一次安装可能较慢；
- 建议使用带 NVIDIA GPU 的机器；
- 如果安装失败，Hermes 应说明失败原因，不要假装成功；
- 不要用于制作侵权、冒充他人、欺骗身份或违法营销图片。
