# AI图片制作能力使用说明

安装完成后，你可以直接把图片或需求发给 Hermes。

## 常见用法

### 生成图片

```text
帮我生成一张封面图，主题是 AI 教育，风格高级、有科技感，标题是：让备课变成作品。
```

### 商品宣传图

```text
我发一张商品图，请帮我做成商品宣传海报。商品本体、Logo、颜色不要变形，背景可以高级一点。
```

### 换背景

```text
把这张图换成高级商业摄影背景，主体不要变形。
```

### 高清放大

```text
把这张图片放大变清晰，适合发朋友圈、社交平台或宣传场景。
```

### 活动海报

```text
帮我做一张活动海报，标题是……，时间是……，地点是……，风格要专业。
```

## 建议提供的信息

- 图片用途：商品图、海报、封面图、朋友圈、公众号头图、社交平台等；
- 尺寸：如果不知道，可以不写，Hermes 会自动判断；
- 标题和文案：需要放在图上的文字；
- 必须保持不变的内容：商品、人物、Logo、包装、颜色等。

## 注意

请不要让它制作冒充他人、侵权或欺骗身份的图片。
