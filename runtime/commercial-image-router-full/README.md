# 图片处理能力安装包

这是一个给普通用户使用的安装包。安装完成后，Hermes/Harness 会获得更强的图片处理能力，可以用于：

- 生成高质量图片
- 做封面图
- 做商品宣传图
- 做活动海报
- 根据真人照片做卡通头像
- 给图片换背景
- 把图片放大变清晰
- 基础擦除/改图

## 你需要做什么

只需要执行一个安装脚本。

```bash
bash 一键安装.sh
```

安装过程会比较久，因为它会自动下载图片模型。正常情况可能需要 30 分钟到数小时，取决于网络速度。

## 安装完成后怎么用

打开 Hermes/Harness，对它说：

```text
/skill commercial-image-router
```

然后就可以直接说需求，例如：

```text
帮我生成一张蜻蜓点水的图片
```

```text
我发你一张商品图，帮我做成宣传海报
```

```text
把这张图放大成高清版
```

```text
根据这张照片做一个微信头像，要像本人，但更卡通
```

## 安装包会自动安装什么

你不用理解这些，只需要知道它们是为了保证图片质量：

- 图片生成工具
- 高质量图片模型
- 商品抠图工具
- 中文字体
- 海报排版工具
- 图片放大工具
- 安装后的测试出图

## 如果安装失败

最常见原因是网络下载模型失败。可以重新执行：

```bash
bash 一键安装.sh
```

如果网络访问 huggingface 镜像很慢，可以执行：

```bash
HF_ENDPOINT=https://hf-mirror.com bash 一键安装.sh
```

## 电脑要求

建议使用带 NVIDIA 显卡的电脑或服务器。没有显卡也可能安装成功，但生成图片会很慢，效果也可能不稳定。
